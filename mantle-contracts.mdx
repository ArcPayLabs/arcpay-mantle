## Contract Roles

- `AgentRegistry`: service metadata, capability tags, endpoint URL, MNT price, and active status.
- `AgentOrderBook`: escrowed order state machine for agent work.
- `TreasuryPolicy`: spend caps, approval thresholds, allowed tokens, allowlist, and pause state.
- `AgentSpendCardVault`: bounded USDY agent spend cards.
- `MantlePrivacyVault`: commitment-first privacy intents.
- `AgentInvoiceBook`: MNT and USDY invoice lifecycle.
- `AgentReputationBook`: order-backed agent scores, completions, disputes, and evidence URIs.
- `OperatorControls`: claim-code onboarding and webhook circuit-breaker controls.
- `MantleAgentRiskOracle`: testnet risk request and callback evidence.
